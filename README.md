# style-transfer

A simple tensorflow implementation of style transfer as outlined in ["A Neural Algorithm of Artistic Style"](https://arxiv.org/abs/1508.06576).  

## Requirements:

Tensorflow  
Pillow (PIL)  
Scipy  
numpy
